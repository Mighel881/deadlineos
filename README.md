# ⚡ DeadlineOS — AI Crisis Command for Serious Productivity

> *The first browser OS for crisis management. AI analyzes your tasks, deadlines and mood — and decides with you. No setup. No account.*

![DeadlineOS Banner](https://deadlineos.app/og-image.png)

## 🚀 What is DeadlineOS?

**DeadlineOS** is a fully browser-based productivity OS that combines AI-powered task management, crisis detection, mood tracking, and gamification — all without requiring a single login or installation.

Built for people who don't just want *another* to-do app. Built for people who are **in the middle of it**.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🤖 **AI Commander** | Chat with an AI that analyzes your tasks, deadlines and stress level — and gives you a real action plan |
| 🚨 **Crisis Radar** | Real-time detection of overdue tasks, deadline collisions and team bottlenecks |
| 🎮 **Gamification** | XP system, streaks, level-ups and achievement badges to keep you motivated |
| 📊 **Analytics Dashboard** | Visual overview of your productivity patterns, team performance and deadline history |
| 🧠 **Mood Tracking** | Daily mood check-ins that influence your AI recommendations |
| 🍅 **Pomodoro Timer** | Built-in focus timer integrated with your task list |
| 📋 **Priority Matrix** | Eisenhower matrix for instant task prioritization |
| 👥 **Team View** | Manage team members, assign tasks and track workload |
| 🔗 **Integrations** | Connect with Slack, GitHub, Jira, Notion, Google Calendar and more |
| 🌙 **4 Color Themes** | Dark UI with scanlines/grain effect — choose your vibe |

## 🛠️ Tech Stack

- **Pure HTML + CSS + JavaScript** — zero dependencies, zero build step
- **AI Integration** — Claude 3.5 Sonnet (Anthropic) via API
- **No backend required** — runs entirely in the browser
- **No account needed** — data stored in localStorage

## 🎯 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/deadlineos.git
   cd deadlineos
   ```

2. **Open in browser**
   ```bash
   open index.html
   # or simply double-click index.html
   ```

3. **Add your API key** *(optional — for AI features)*
   - Open `deadlineos-v3.html`
   - Enter your [Anthropic API key](https://console.anthropic.com/) in the settings panel

> No npm install. No webpack. No server. Just open and go. ⚡
