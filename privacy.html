🔒 Privacy

YOUR DATA.
YOUR CONTROL.

DeadlineOS is built so that your data never touches our servers.

⚡ The Short Version
All data is stored locally.

What We Store

DeadlineOS stores the following data exclusively locally:

  • Tasks, priorities, completion status
  • Notes and daily plan entries
  • Mood, Energy, mood-tracking data
  • XP, level, achievement status
  • AI chat history (letzte 8 Nachrichten)
  • Prole names and PINs (hashed in browser)
  • Timer-Sessions und Focus statistics
  • Farbtheme und UI-Settings

This data does not leave your browser.

AI features und Anthropic API

Wenn du AI features nutzt, wird eine Anfrage an die Anthropic API gesendet.

  • Your task list (text, priorities)
  • Dein aktuelles Stress-Level und Energy level
  • Deine Mood
  • The chat history of the current session

This data is subject to Anthropic's Privacy Policy.

Your API key is only used in your browser.

Cookies

DeadlineOS does not use cookies.

Third parties

  • Google Fonts CDN
  • Anthropic API

Keine weiteren Third parties.

Data deletion

You can delete your data yourself at any time:

  • Browser → Settings → Delete website data
  • Oder in der App: Profile → Settings → Delete all data

There is no data on our servers.

Children

DeadlineOS is intended for users aged 16 and over.

Changes

If this Privacy Policy changes, we will update the date below.
