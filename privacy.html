🔒 Datenschutz

DEINE DATEN.
DEINE KONTROLLE.

DeadlineOS wurde so gebaut, dass deine Daten niemals unsere Server berühren.

⚡ Die Kurzfassung
Alle Daten (Tasks, Notizen, Einstellungen, Chat-Verlauf) werden ausschließlich in deinem Browser gespeichert — via localStorage. Wir haben keinen Server, der deine Daten kennt. Die einzige externe Verbindung ist die Anthropic API für KI-Features — und auch dort werden keine personenbezogenen Daten übertragen.

Was wir speichern

DeadlineOS speichert folgende Daten ausschließlich lokal in deinem Browser (localStorage):

  • Tasks, Prioritäten, Erledigungsstatus
  • Notizen und Tagesplaneinträge
  • Stimmung, Energie, Moodtracking-Daten
  • XP-Stand, Level, Achievement-Status
  • KI-Chat-Verlauf (letzte 8 Nachrichten)
  • Profil-Namen und PINs (gehasht im Browser)
  • Timer-Sessions und Fokus-Statistiken
  • Farbtheme und UI-Einstellungen

Diese Daten verlassen deinen Browser nicht. Sie werden nicht an uns übermittelt, nicht analysiert, nicht verkauft.

KI-Features und Anthropic API

Wenn du KI-Features nutzt (KI-Analyse, KI-Chat, KI-Tagesplan), wird eine Anfrage direkt von deinem Browser an die Anthropic API gesendet. Diese Anfrage enthält:

  • Deine Task-Liste (Texte, Prioritäten)
  • Dein aktuelles Stress-Level und Energieniveau
  • Deine Stimmung (als generische Bezeichnung wie "gut" oder "gestresst")
  • Den Chat-Verlauf der aktuellen Sitzung

Diese Daten werden an Anthropic übermittelt und unterliegen der Datenschutzerklärung von Anthropic. Wir empfehlen, keine echten Namen oder sensiblen Informationen in Tasks einzutragen, die du nicht mit einem KI-Dienst teilen möchtest.

Dein API-Key wird nur in deinem Browser verwendet. Er wird nicht an uns übertragen.

Cookies

DeadlineOS verwendet keine Cookies. Es gibt keine Tracking-Pixel, keine Analytics-Scripts, keine Werbung.

Drittanbieter

  • Google Fonts CDN — Lädt Schriftarten (Bebas Neue, Lora, DM Mono). Google kann dabei die IP-Adresse des Browsers sehen. Alternativ können die Fonts lokal gehostet werden.
  • Anthropic API — Nur wenn KI-Features aktiv genutzt werden (siehe oben).

Keine weiteren Drittanbieter. Kein Facebook, kein Google Analytics, kein Hotjar, kein Intercom.

Datenlöschung

Da alle Daten lokal gespeichert sind, kannst du sie jederzeit selbst löschen:

  • Browser → Einstellungen → Websitedaten löschen → deadlineos.app
  • Oder in der App: Profil → Einstellungen → Alle Daten löschen

Es gibt keine Daten auf unseren Servern, die gelöscht werden müssten.

Kinder

DeadlineOS richtet sich an Nutzerinnen und Nutzer ab 16 Jahren. Wir sammeln wissentlich keine Daten von Kindern unter 16 Jahren.

Änderungen

Wenn sich diese Datenschutzerklärung ändert, aktualisieren wir das Datum unten. Da wir keine E-Mail-Adressen sammeln, können wir dich nicht proaktiv informieren — bitte schau gelegentlich vorbei.
